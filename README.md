## How-to install:

- Install Vim-plug

``` 
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

- Run the following command inside vim

```
:PlugInstall
```

- Exit vim

```
:q
```

- Re-enter vim
